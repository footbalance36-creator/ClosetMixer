06

ViewModels — pattern réel

Pas d'AndroidX ViewModel. Chaque ViewModel gère son propre scope de coroutine :

class XxxViewModel(...) {
    private val scope = CoroutineScope(SupervisorJob() + Dispatchers.Main)
    private val _uiState = MutableStateFlow(XxxUiState())
    val uiState: StateFlow<XxxUiState> = _uiState.asStateFlow()
}

Les ViewModels sont enregistrés en single dans Koin — une seule instance par ViewModel pour toute la session.

API Météo — Open-Meteo

✅
Sans clé API — entièrement gratuit Contrairement au spec initial qui prévoyait OpenWeatherMap, l'appli utilise Open-Meteo. 
GET https://api.open-meteo.com/v1/forecast
  ?latitude=...&longitude=...
  ¤t=temperature_2m,weathercode,relative_humidity_2m
  &timezone=auto

Réponse mappée dans WeatherDto / CurrentWeatherDto. Codes météo traduits en français et en emoji dans WeatherDto.kt.
